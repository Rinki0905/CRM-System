# CRM Pro: Full-Stack AI-Powered CRM
CRM Pro is a complete full-stack MERN application built from scratch. It serves as a modern, responsive, and intelligent Customer Relationship Management (CRM) system designed to help businesses manage contacts, track sales pipelines, and organize tasks. The application features a secure backend API, a dynamic React frontend, and an integrated AI assistant.

## [https://crm-frontend-e5cj.onrender.com]
## Key Features
* Secure Authentication: Full user registration and login system using JWT (JSON Web Tokens) and bcrypt for password hashing.

* Dynamic Dashboard: A real-time dashboard that aggregates and displays key metrics, including total customers, active deals, and total revenue.

* Customer Management: Full CRUD (Create, Read, Update, Delete) functionality for managing customer profiles and contact information.

* Interactive Deal Pipeline: A drag-and-drop Kanban board (built with dnd-kit) to visually manage sales deals through various stages (Lead, Proposal, Won, etc.).

* Task Management: A complete to-do system to create, update, delete, and filter tasks by status (Pending, Completed).

* AI Assistant: An integrated chatbot powered by the Google Gemini API to provide users with intelligent assistance.

* Fully Responsive: The UI is built with Tailwind CSS and is fully responsive, providing a seamless experience on both desktop and mobile devices.
